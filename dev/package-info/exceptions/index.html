Exceptions

Semigroups.Errors.LibsemigroupsError — Type
LibsemigroupsError <: Exception

Exception type for errors originating from the libsemigroups C++ library. The error message has the C++ source location prefix stripped but is otherwise unchanged (including 0-based indexing).

source
Semigroups.Errors.@wrap_libsemigroups_call — Macro
@wrap_libsemigroups_call(expr)

Wrap a libsemigroups C++ call to catch exceptions and rethrow them as LibsemigroupsError with the C++ prefix stripped.

Warning

Error messages originate from the C++ library and use 0-based indexing for positions and values.

Example

cxx_obj = @wrap_libsemigroups_call begin
    CxxType(StdVector(images_typed))
end
