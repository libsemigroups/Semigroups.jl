Full API

Semigroups.Transf — Type
Transf

A transformation is a function from {1, 2, ..., n} to itself. The transformation is represented by a vector of images.

Construction

# Create transformation [1, 2, 1] (maps 1->1, 2->2, 3->1)
t = Transf([1, 2, 1])

Note: The constructor accepts 1-based indexing (Julia convention) but internally converts to 0-based indexing for the C++ library.

source

Construction

# Create a transformation from images (1-based indexing)
t = Transf([2, 3, 1, 4])  # maps 1→2, 2→3, 3→1, 4→4

# The degree is inferred from the length
degree(t)  # 4

Indexing

Transformations support indexing to get individual images:

t = Transf([2, 3, 1])
t[1]  # 2
t[2]  # 3
t[3]  # 1

Composition

Transformations can be composed using *:

s = Transf([2, 1, 3])
t = Transf([3, 2, 1])
s * t  # apply s first, then t
