Transf, PPerm, and Perm Helpers

This page contains the documentation for helper functions that work with transformations, partial permutations, and permutations.

Semigroups.image_set — Function
image_set(t::Transf) -> Vector{Int}

Return a sorted vector of all distinct values in the image of t. Uses 1-based indexing.

source
image_set(p::PPerm) -> Vector{Int}

Return a sorted vector of all points in the image of p. Uses 1-based indexing.

source
image_set(p::Perm) -> Vector{Int}

Return a sorted vector of all distinct values in the image of p. For permutations, this is always [1, 2, ..., degree(p)]. Uses 1-based indexing.

source
Semigroups.domain_set — Function
domain_set(p::PPerm) -> Vector{Int}

Return a sorted vector of all points in the domain of p (where p is defined). Uses 1-based indexing.

source
Semigroups.left_one — Function
left_one(p)

Return the partial permutation that acts as the identity on the domain of p. This is the partial permutation that maps each point in domain(p) to itself.

source
left_one(p::PPerm) -> PPerm

Return the partial permutation that acts as identity on the domain of p.

source
Semigroups.right_one — Function
right_one(p)

Return the partial permutation that acts as the identity on the image of p. This is the partial permutation that maps each point in image(p) to itself.

source
right_one(p::PPerm) -> PPerm

Return the partial permutation that acts as identity on the image of p.

source

Examples

Degree and Rank

t = Transf([2, 2, 3])
degree(t)  # 3 (size of the domain)
rank(t)    # 2 (number of distinct image points: {2, 3})

Identity Elements

t = Transf([2, 3, 1, 4])

# Left identity: e * t == t
e = left_one(t)   # identity transformation of same degree

# Right identity: t * e == t
r = right_one(t)  # identity on image set
