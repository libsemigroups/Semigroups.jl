Perm

A permutation is a bijective function from {1, 2, ..., n} to itself.

Construction

# Create permutation [2, 3, 1] (maps 1->2, 2->3, 3->1)
p = Perm([2, 3, 1])

The constructor validates that the input is a valid permutation.

source

Construction

# Create from images (must be a bijection)
p = Perm([2, 3, 1])  # cycle (1 2 3)

# All points must be mapped
degree(p)  # 3

Inverse

Permutations can be inverted:

p = Perm([2, 3, 1])
inv(p)  # Perm([3, 1, 2])

p * inv(p)  # identity

Cycle Notation

The permutation Perm([2, 3, 1]) represents the cycle $(1\ 2\ 3)$, meaning:

  • 1 maps to 2
  • 2 maps to 3
  • 3 maps to 1
