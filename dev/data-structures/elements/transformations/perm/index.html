The Perm Type

A permutation $f$ is an injective transformation defined on the whole of $\{1, 2, \ldots, n\}$ for some integer $n$ called the degree of $f$. A permutation is stored as a vector of the images of $\{1, 2, \ldots, n\}$, i.e. $((1)f, (2)f, \ldots, (n)f)$.

Semigroups.Perm — Type
Perm{T}

Permutations with dynamic degree.

A permutation $f$ is an injective transformation defined on the whole of $\{1, 2, \ldots, n\}$ for some integer $n$ called the degree of $f$. A permutation is stored as a vector of the images of $\{1, 2, \ldots, n\}$, i.e. $((1)f, (2)f, \ldots, (n)f)$.

Construction

using Semigroups

# From an image list
p = Perm([2, 3, 1])  # maps 1->2, 2->3, 3->1
p[1]  # 2
p[2]  # 3

Semigroups.degree — Method
degree(p::Perm) -> Int

Returns the degree of a permutation.

The degree of a permutation is the number of points used in its definition, which is equal to the size of its underlying container.

Example

julia> using Semigroups

julia> degree(Perm([2, 3, 1]))
3

Semigroups.domain — Method
domain(f::Union{Transf, PPerm, Perm}) -> Vector{Int}

Return the sorted set of points where a partial transformation is defined.

Returns a vector containing those values $i$ such that $i \in \{1, \ldots, n\}$ where $n$ is the degree of f, and f[i] != UNDEFINED.

Complexity

$O(n)$ where $n$ is the degree of f.

See also image.

Example

julia> using Semigroups

julia> domain(Transf([2, 2, 1]))
3-element Vector{Int64}:
 1
 2
 3

julia> domain(PPerm([1, 3], [2, 4], 5))
2-element Vector{Int64}:
 1
 3

Semigroups.image — Method
image(f::Union{Transf, PPerm, Perm}) -> Vector{Int}

Return the sorted set of image values of a partial transformation.

Returns a vector containing those values f[i] such that $i \in \{1, \ldots, n\}$ where $n$ is the degree of f, and f[i] != UNDEFINED.

Complexity

$O(n \log n)$ where $n$ is the degree of f.

See also domain.

Example

julia> using Semigroups

julia> image(Transf([2, 2, 1]))
2-element Vector{Int64}:
 1
 2

julia> image(PPerm([1, 3], [2, 4], 5))
2-element Vector{Int64}:
 2
 4

Semigroups.inverse — Method
inverse(p::T) where T<:Union{PPerm,Perm} -> T

Returns the inverse of a partial permutation or permutation.

This function returns a newly constructed inverse of p. The inverse of a partial permutation p is the partial term g such that fgf = f and gfg =g.

Example

julia> using Semigroups

julia> p = Perm([2, 3, 1]);

julia> inverse(p)
Perm([3, 1, 2])

julia> p * inverse(p) == one(Perm, 3)
true

Base.one — Method
Base.one(::Type{Perm}, n::Integer) -> Perm

Returns the identity permutation on n points.

This function returns a newly constructed permutation with degree equal to n that fixes every value from 1 to n.

Example

julia> using Semigroups

julia> one(Perm, 3)
Perm([1, 2, 3])

Semigroups.rank — Method
rank(p::Perm) -> Int

Returns the number of distinct image values of a permutation.

The rank of a permutation is the number of its distinct image values, not including UNDEFINED.

Example

julia> using Semigroups

julia> rank(Perm([2, 3, 1]))

Complexity

Linear in degree()

source

Construction

using Semigroups

# From a vector of images (must be a bijection)
p = Perm([2, 3, 1])  # maps 1->2, 2->3, 3->1
p[1]  # 2

# The degree is inferred from the length
degree(p)  # 3

Inverse

p = Perm([2, 3, 1])
q = inverse(p)       # Perm([3, 1, 2])
p * inverse(p) == one(Perm, 3)  # true

Composition

Permutations can be composed using *:

p = Perm([2, 3, 1])
q = Perm([3, 1, 2])
r = p * q  # Perm([1, 2, 3]) — the identity

Comparison

Permutations support equality and lexicographic ordering:

p = Perm([2, 3, 1])
q = copy(p)
p == q  # true
p < Perm([3, 1, 2])  # true
