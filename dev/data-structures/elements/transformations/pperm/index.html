The PPerm Type

A partial permutation $f$ is an injective partial transformation, which is stored as the vector of images of $\{1,2,\ldots,n\}$, i.e. $((1)f,(2)f,\ldots,(n)f)$ where the value UNDEFINED is used to indicate that $(i)f$ is undefined (i.e. not among the points where $f$ is defined).

Semigroups.PPerm — Type
PPerm{T}

Partial permutations with dynamic degree.

A partial permutation $f$ is just an injective partial transformation, which is stored as a vector of the images of $\{1, 2, \ldots, n\}$, i.e. $((1)f, (2)f, \ldots, (n)f)$ where the value UNDEFINED is used to indicate that $(i)f$ is undefined (i.e. not among the points where $f$ is defined).

Construction

using Semigroups

# From an image list (use UNDEFINED for undefined points)
p = PPerm([2, UNDEFINED, 1])
p[1]  # 2
p[2]  # UNDEFINED

# From domain, image, and degree
p = PPerm([1, 3], [2, 1], 3)  # 1 -> 2, 3 -> 1, degree 3

Semigroups.degree — Method
degree(p::PPerm) -> Int

Returns the degree of a partial permutation.

The degree of a partial permutation is the number of points used in its definition, which is equal to the size of its underlying container.

Example

julia> using Semigroups

julia> degree(PPerm([1, 3], [2, 4], 5))
5

Semigroups.domain — Method
domain(f::Union{Transf, PPerm, Perm}) -> Vector{Int}

Return the sorted set of points where a partial transformation is defined.

Returns a vector containing those values $i$ such that $i \in \{1, \ldots, n\}$ where $n$ is the degree of f, and f[i] != UNDEFINED.

Complexity

$O(n)$ where $n$ is the degree of f.

See also image.

Example

julia> using Semigroups

julia> domain(Transf([2, 2, 1]))
3-element Vector{Int64}:
 1
 2
 3

julia> domain(PPerm([1, 3], [2, 4], 5))
2-element Vector{Int64}:
 1
 3

Semigroups.image — Method
image(f::Union{Transf, PPerm, Perm}) -> Vector{Int}

Return the sorted set of image values of a partial transformation.

Returns a vector containing those values f[i] such that $i \in \{1, \ldots, n\}$ where $n$ is the degree of f, and f[i] != UNDEFINED.

Complexity

$O(n \log n)$ where $n$ is the degree of f.

See also domain.

Example

julia> using Semigroups

julia> image(Transf([2, 2, 1]))
2-element Vector{Int64}:
 1
 2

julia> image(PPerm([1, 3], [2, 4], 5))
2-element Vector{Int64}:
 2
 4

Semigroups.inverse — Method
inverse(p::T) where T<:Union{PPerm,Perm} -> T

Returns the inverse of a partial permutation or permutation.

This function returns a newly constructed inverse of p. The inverse of a partial permutation p is the partial term g such that fgf = f and gfg =g.

Example

julia> using Semigroups

julia> p = Perm([2, 3, 1]);

julia> inverse(p)
Perm([3, 1, 2])

julia> p * inverse(p) == one(Perm, 3)
true

Semigroups.left_one — Method
left_one(p::PPerm) -> PPerm

Returns the left one of a partial permutation.

This function returns a newly constructed partial permutation with degree equal to that of p that fixes every value in the domain of p, and is UNDEFINED on any other values.

Example

julia> using Semigroups

julia> p = PPerm([1, 3], [2, 4], 4);

julia> left_one(p) * p == p
true

Semigroups.rank — Method
rank(p::PPerm) -> Int

Returns the number of distinct image values in a partial permutation.

The rank of a partial permutation is the number of its distinct image values, not including UNDEFINED.

Example

julia> using Semigroups

Complexity

Linear in degree()

source

Construction

using Semigroups

# From a vector of images (use UNDEFINED for undefined points)
p = PPerm([3, 4, UNDEFINED, UNDEFINED, UNDEFINED])
p[1]  # 3
p[3]  # UNDEFINED

# From domain, range, and degree
p = PPerm([1, 2], [3, 4], 5)  # 1 -> 3, 2 -> 4, degree 5

Inverse

p = PPerm([1, 2], [3, 4], 5)
q = inv(p)           # maps 3 -> 1, 4 -> 2
p * inv(p) == left_one(p)   # true
inv(p) * p == right_one(p)  # true

Left and Right Ones

p = PPerm([1, 3], [2, 4], 4)
left_one(p) * p == p   # true
p * right_one(p) == p  # true

Composition

Partial permutations can be composed using *:

p = PPerm([1, 2], [3, 4], 5)
q = PPerm([3, 4], [5, 1], 5)
r = p * q  # 1 -> 5, 2 -> 1

Comparison

Partial permutations support equality and lexicographic ordering:

p = PPerm([1, 2], [3, 4], 5)
q = copy(p)
p == q  # true
p < PPerm([1, 2], [4, 3], 5)  # true
