The WordRange type

This page contains the documentation of the type WordRange, an iterable range over words in an alphabet in a user-selected Order.

Semigroups.WordRange — Type
WordRange

An iterable range over words in an alphabet, in a user-selected order.

Configure with set_alphabet_size!, set_min!, set_max!, set_first!, set_last!, set_order!, set_upper_bound!, then iterate with Base.get(::WordRange) / next! / at_end or the standard Julia iteration protocol (for w in r, collect(r), etc.).

A default-constructed WordRange is empty, with alphabet size 0, order ORDER_SHORTLEX, and empty first/last bookends.

Example

julia> using Semigroups

julia> r = WordRange();

julia> set_alphabet_size!(r, 2);

julia> set_min!(r, 1);

julia> set_max!(r, 3);

julia> collect(r)
6-element Vector{Vector{Int64}}:
 [1]
 [2]
 [1, 1]
 [1, 2]
 [2, 1]
 [2, 2]

Base.count — Method
count(r::WordRange) -> Int

The actual size of the range.

Returns the number of words in a WordRange object. If the order is ORDER_SHORTLEX then size_hint is used. If the order is not shortlex then a copy of the range may have to be looped over in order to compute the return value.

Semigroups.set_first! — Method
set_first!(r::WordRange, w::AbstractVector{<:Integer}) -> WordRange

Set the first word (lower bookend) of r to w.

Performs no checks on its arguments. If w contains letters greater than alphabet_size, then r will be empty. Similarly, if the first word is greater than the last word with respect to the current order, r will be empty.

Arguments

Semigroups.set_max! — Method
set_max!(r::WordRange, n::Integer) -> WordRange

Set one past the maximum word length of r to n.

Sets the one-past-the-last word of r to the word consisting of n copies of letter 1. So, after set_max!(r, n), r contains words of length strictly less than n.

Arguments

  • n::Integer: one greater than the maximum length.

Semigroups.valid — Method
valid(r::WordRange) -> Bool

Return whether the settings have not changed since the last call to next! or Base.get(::WordRange).

Other than by calling next!, the value returned by Base.get(::WordRange) may be altered by a call to any of: set_order!, set_alphabet_size!, set_min!, set_max!, set_first!, set_last!, set_upper_bound!.

Returns true if none of these settings have changed since the last next! or Base.get(::WordRange) call, and false otherwise.

source
Semigroups.number_of_words — Method
number_of_words(n::Integer, min::Integer, max::Integer) -> Int

Return the number of words over an alphabet of n letters with length in the range [min, max), i.e. $\sum_{i=\min}^{\max-1} n^i$.

Arguments

  • n::Integer: the number of letters in the alphabet.
  • min::Integer: the minimum word length.
  • max::Integer: one greater than the maximum word length.
Warning

If the resulting count exceeds 2^64 - 1 the return value will not be correct.
