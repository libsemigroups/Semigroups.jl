The FroidurePin type

This page documents the parametric type FroidurePin{E}, which implements the Froidure-Pin algorithm for enumerating the elements of a finitely generated semigroup.

FroidurePin{E} is a subtype of Runner (via the internal FroidurePinBase), so all runner methods (run!, run_for!, finished, etc.) are available.

v1 limitation

Semigroups.jl v1 binds FroidurePin{E} for the element types Transf, PPerm, Perm, and BMat8 only.

Semigroups.FroidurePin — Type
FroidurePin{E}

Type implementing the Froidure-Pin algorithm.

A FroidurePin{E} instance is defined by a generating set of elements of type E, and the main function is run!, which implements the Froidure-Pin algorithm as described by Froidure and Pin. If run! is invoked and finished returns true, then the size length, the left and right Cayley graphs left_cayley_graph and right_cayley_graph are determined, and a confluent terminating presentation for the semigroup is known.

Supported element types:

  • Transf{UInt8}, Transf{UInt16}, Transf{UInt32} (transformations)
  • PPerm{UInt8}, PPerm{UInt16}, PPerm{UInt32} (partial permutations)
  • Perm{UInt8}, Perm{UInt16}, Perm{UInt32} (permutations)
  • BMat8 (boolean matrices up to 8x8)

Example

using Semigroups

S = FroidurePin(Transf([2, 1, 3]), Transf([2, 3, 1]))
length(S)  # 6 (symmetric group S_3)

Semigroups.FroidurePin — Method
FroidurePin(gens::Vector{E}) where {E}

Construct a FroidurePin{E} from a container of generators.

This function constructs a FroidurePin{E} instance from the vector of generators gens, after verifying that the proposed generators all have equal degree.

Arguments

  • gens::Vector{E}: a non-empty vector of generators. All generators must have the same degree.

Throws

  • ErrorException: if gens is empty.
  • LibsemigroupsError: if degree(x) != degree(y) for any x and y in gens.

Example

using Semigroups

S = FroidurePin([Transf([2, 1, 3]), Transf([2, 3, 1])])
length(S)  # 6

Semigroups.enumerate! — Method
enumerate!(fp::FroidurePin, limit::Integer) -> FroidurePin

Enumerate until at least a specified number of elements are found.

If fp is already fully enumerated, or the number of elements previously enumerated exceeds limit, then calling this function does nothing. Otherwise, run! attempts to find at least the maximum of limit and batch_size additional elements of the semigroup.

Arguments

  • fp::FroidurePin: the FroidurePin instance.
  • limit::Integer: the approximate limit for current_size(fp).

Complexity

At worst $O(m n)$ where $m$ equals limit and $n$ is number_of_generators(fp).

Example

using Semigroups

S = FroidurePin(Transf([2, 1, 3]), Transf([2, 3, 1]))
enumerate!(S, 4)
current_size(S)  # >= 4

  • current_size

Semigroups.batch_size — Method
batch_size(fp::FroidurePin) -> Int

Return the current value of the batch size.

The batch size is the minimum number of new elements to be found by any call to run!. This is used by, for example, position so that it is possible to find the position of an element after only partially enumerating the semigroup.

The default value of the batch size is 8192.

Complexity

Constant.

Base.iterate — Method
Base.iterate(fp::FroidurePin{E}[, state]) -> Union{Tuple{E, Int}, Nothing}

Iterate over all elements of the semigroup.

This function allows a FroidurePin{E} instance to be used in for loops and with collect. Elements are yielded in the order they were enumerated by the Froidure-Pin algorithm.

Note

This function triggers a full enumeration on the first call (via length).

Example

S = FroidurePin(Transf([2, 1, 3]), Transf([2, 3, 1]))
for x in S
    println(x)
end
elts = collect(S)  # Vector{Transf{UInt8}} of length 6

  • length
