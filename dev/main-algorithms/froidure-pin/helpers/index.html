Helper functions

This page collects the free functions that operate on a FroidurePin instance. They mirror the libsemigroups::froidure_pin::* namespace and are organised into four groups: factorisations, collections, word-element conversion, and Cayley graphs.

Table of contents

SectionDescription
FactorisationsMinimal and non-minimal factorisations of elements as generator-index words.
CollectionsMaterialized vectors of rules, normal forms, idempotents, and sorted elements.
Word-element conversionConvert between generator-index words and semigroup elements.
Cayley graphsLeft and right Cayley graphs as WordGraph objects.

Factorisations

These functions return or query words (as Vector{Int} with 1-based generator indices) representing elements of the semigroup.

Contents

FunctionDescription
minimal_factorisationMinimal factorisation of the element at a given position.
current_minimal_factorisationMinimal factorisation without triggering enumeration.
factorisationA (not necessarily minimal) factorisation.

Full API

Semigroups.minimal_factorisation — Method
minimal_factorisation(fp::FroidurePin, i::Integer) -> Vector{Int}

Return the minimal factorisation of the element at 1-based position i as a 1-based generator-index word.

Example

S = FroidurePin(Transf([2, 1, 3]), Transf([2, 3, 1]))
w = minimal_factorisation(S, 1)  # [1] or [2] — a single generator
source
Semigroups.current_minimal_factorisation — Method
current_minimal_factorisation(fp::FroidurePin, i::Integer) -> Vector{Int}

Return the minimal factorisation of the element at 1-based position i without triggering further enumeration.

source
Semigroups.factorisation — Method
factorisation(fp::FroidurePin, i::Integer) -> Vector{Int}

Return the factorisation of the element at 1-based position i as a 1-based generator-index word.

This may not be the minimal factorisation.

source

Full API

Semigroups.rules — Method
rules(fp::FroidurePin) -> Vector{Pair{Vector{Int}, Vector{Int}}}

Return all defining rules of the semigroup as a vector of lhs => rhs pairs, where each side is a 1-based generator-index word.

Triggers full enumeration if not already complete.

Example

S = FroidurePin(Transf([2, 1, 3]), Transf([2, 3, 1]))
rs = rules(S)
for (lhs, rhs) in rs
    println(lhs, " => ", rhs)
end
source

Semigroups.idempotents — Method
idempotents(fp::FroidurePin{E}) -> Vector{E}

Return all idempotent elements of the semigroup (elements x such that x * x == x).

Triggers full enumeration if not already complete.

Example

S = FroidurePin(Transf([2, 1, 3]), Transf([2, 3, 1]))
ids = idempotents(S)  # [Transf([1, 2, 3])]
source
Semigroups.sorted_elements — Method
sorted_elements(fp::FroidurePin{E}) -> Vector{E}

Return all elements of the semigroup in sorted order.

Triggers full enumeration if not already complete.

Example

S = FroidurePin(Transf([2, 1, 3]), Transf([2, 3, 1]))
se = sorted_elements(S)

Word-element conversion

These functions convert between generator-index words (Vector{Int} with 1-based indices) and semigroup elements.

Contents

FunctionDescription
to_elementConvert a word to the corresponding element.

Full API

Semigroups.to_element — Method
to_element(fp::FroidurePin{E}, w::AbstractVector{<:Integer}) -> E

Convert a 1-based generator-index word w to the corresponding element.

Example

S = FroidurePin(Transf([2, 1, 3]), Transf([2, 3, 1]))
x = to_element(S, [1, 2])  # product of generators 1 and 2
source
Semigroups.equal_to — Method
equal_to(fp::FroidurePin, w1::AbstractVector{<:Integer}, w2::AbstractVector{<:Integer}) -> Bool

Check whether two 1-based generator-index words represent the same element.

Example

S = FroidurePin(Transf([2, 1, 3]), Transf([2, 3, 1]))
equal_to(S, [1, 1], [1])  # does gen1*gen1 == gen1?
source
Semigroups.position — Method
position(fp::FroidurePin, w::AbstractVector{<:Integer}) -> Int

Return the 1-based position of the element represented by the 1-based generator-index word w.

Triggers full enumeration if not already complete.

Example

S = FroidurePin(Transf([2, 1, 3]), Transf([2, 3, 1]))
position(S, [1])  # position of generator 1
