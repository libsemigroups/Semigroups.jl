Overview

This section documents the main algorithms available in Semigroups.jl.

Available Algorithms

Froidure-Pin Algorithm

The Froidure-Pin algorithm enumerates all elements and determines the structure of a finitely generated semigroup or monoid. See the Froidure-Pin section for the full API.

Planned Algorithms

The following algorithms from libsemigroups will be exposed in future versions of Semigroups.jl:

  • Knuth-Bendix Completion
  • Todd-Coxeter Algorithm
  • Kambites' Algorithm (small overlap monoids)
  • Schreier-Sims Algorithm (permutation groups)
